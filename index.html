Dicas e Atalhos Úteis

Utilize os atalhos abaixo no seu teclado para programar mais rápido:

  • Ctrl + C : Copiar texto selecionado
  • Ctrl + X : Recortar (Remove e copia)
  • Ctrl + Z : Desfazer a última ação
  • Ctrl + Y : Refazer ação
  • Ctrl + A : Selecionar TODO o código

💡 Atenção: A função de colar (Ctrl+V) está desativada no sistema para garantir que você pratique a digitação do código! Copiar (Ctrl+C) está liberado para o envio das atividades.

⚠️ Se o seu código tiver algum erro em inglês ao executar, clique no botão amarelo "Traduzir Erro" que aparecerá no Terminal!
